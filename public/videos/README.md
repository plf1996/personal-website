# 视频资源说明

这个文件夹用于存放网站的背景视频文件。

## 推荐的视频文件：

### 1. 主背景视频
- **文件名**: `background.mp4` 或 `hero-background.mp4`
- **要求**:
  - 分辨率: 1920x1080 或更高
  - 时长: 15-30秒循环
  - 格式: MP4 (H.264)
  - 文件大小: < 20MB
  - 内容: 展现年轻人活力、技术感、现代感的动态背景

### 2. 备用视频文件
- `background-fallback.mp4` - 备用视频源
- `background-mobile.mp4` - 移动设备优化版本（低分辨率）

## 视频内容建议：
- 科技感动画
- 抽象几何图形
- 粒子效果
- 流体动画
- 城市延时摄影
- 代码滚动效果
- 数字雨效果

## 免费视频资源网站：
- Pexels Videos
- Pixabay Videos
- Mixkit
- Videvo
- Coverr

## 使用方法：
将视频文件放在此目录下，然后在 `src/components/BackgroundVideo.tsx` 中更新视频路径。
